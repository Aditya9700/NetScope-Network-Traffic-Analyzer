Quick Start Guide

  1. 1
    Select Network Interface

    Navigate to the Live Monitor page, choose your active network interface (Ethernet/Wi-Fi) from the configuration bar, and click start.

  2. 2
    Generate Traffic

    Open a web browser or terminal, ping external servers or browse sites to trigger TCP, UDP, ICMP, and ARP traffic for live parsing.

  3. 3
    Inspect Packets

    Click on any captured row in the live table to inspect Ethernet headers, IP frames, TCP handshakes, and hexadecimal payloads.

  4. 4
    Analyze PCAP

    Upload files on the PCAP Analyzer page to parse captures created on external routers or Wireshark.
